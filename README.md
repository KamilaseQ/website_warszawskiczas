# Warszawski Czas Website

Standalone Next.js project for the public Warszawski Czas website.

## Requirements

- Node.js 20+
- npm

## Development

```bash
npm install
npm run dev
```

The app runs on `http://localhost:3000` by default.

## Production build

```bash
npm run build
npm run start
```

## Environment

Copy `.env.example` to `.env.local` for local contact-form and private-collection settings.
